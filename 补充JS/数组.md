# 数组相关方法属性

## 1.arr.Map(()=>{})方法

``` javascript
let arr = [1,2,3,4,5];
let arr2 = arr.map((element, index) => {
    return element + 1;
})
console.log(arr2);
//[2,3,4,5,6];
```

1. 参数：**`element`**：数组中每个元素；**`index`**：每个元素的索引下标
2. return element + 1 实际上是返回给()=>{}这个箭头函数，这个函数将返回值存储，最后再返回给arr2变量
3. **`该方法不会改变原数组，返回一个新的数组`**
