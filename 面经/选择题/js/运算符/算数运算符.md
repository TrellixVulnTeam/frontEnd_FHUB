## 1.

如下代码输出的结果是什么：

``` javascript
console.log(1+ "2"+"2");
console.log(1+ +"2"+"2");
console.log("A"- "B"+"2");
console.log("A"- "B"+2);

/* 
	1. console.log(1 + "2" + "2");
	-都是同优先级运算符，所以从左到右依次计算
	-先计算1 + "2", 得到"12"
	-然后计算"12" + "2" 得到 "122";
	-因此输出122
	
	2. console.log(1+ +"2"+"2");
	-注意有一个一元运算符 +"2"，其优先级高于算术运算符
	-先计算一元运算符 +"2"， 相当于是Number()函数，所以得到2
	-式子变为1 + 2 + "2";
	-接着从左到右计算
	-先算1 + 2， 得到3；
	-然后计算3 + "2" 得到"32";
	-因此输出32
	
	3、console.log("A"-"B"+"2");
	-都是同优先级运算符，所以从左到右依次计算
	-先计算"A"-"B"，将两边用Number()函数转换成数字进行计算，Number(A)=NaN,Number(B)=NaN,所以得到NaN
	-然后计算NaN + "2"，得到"NaN2"
	-所以最后输出"NaN2"
	
	4. console.log("A"-"B"-2);
	-都是同优先级运算符，所以从左到右依次计算
	-先计算"A"-"B"，将两边用Number()函数转换成数字进行计算，Number(A)=NaN,Number(B)=NaN,所以得到NaN
	-然后计算NaN + 2，得到NaN
	-最后输出NaN
*/
```
