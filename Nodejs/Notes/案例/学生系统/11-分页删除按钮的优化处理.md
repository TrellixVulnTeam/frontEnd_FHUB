## 删除优化

- 如果`只剩一个数据项`，删除后，`返回上一页`。如果`不止一个数据项`，删除后，`停留在这一页`
- `在数据库成功删除对应数据项后`，在`刷新页面重新渲染列表前`，通过选择器获得`未刷新页面前当前页的数据条数`。如果只`有一条`，全局变量`param.pageIndex--`；`否则停留在这一页`

``` javascript
            if (msg.status) {
                // 获取当前tbody中有多少行数据
                if ($('tbody tr').length == 1) {
                    // 只有一条待删除的数据，删除后到前面一页
                    param.pageIndex--;
                }
                fetchInfo();
            }
        }
```
